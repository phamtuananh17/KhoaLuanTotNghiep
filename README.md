# KhoaLuanTotNghiep
Phần mềm quản lí bãi giữ xe có những chức năng cơ bản như :
  - Quản lý nhân viên
  - Quản lý xe vào
  - Quản lý xe ra
  - Quản lý thẻ xe
  - Quản lý dịch vụ
  - Thống kê, báo cáo
